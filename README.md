# Machine Learning Practice Projects 🚀

This repository contains my Machine Learning practice projects developed using Python and Scikit-learn. These projects helped me improve my skills in data preprocessing, model building, evaluation, and visualization.

## 📂 Projects Included

### 1. Customer Churn Prediction
Predict whether a customer will leave a company or continue using its services.

**Techniques Used:**
- Data Preprocessing
- Label Encoding
- Train-Test Split
- Classification Algorithms
- Model Evaluation

**Goal:** Predict customer churn.

---

### 2. Diabetes Prediction
Predict whether a patient has diabetes based on medical attributes.

**Techniques Used:**
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Selection
- Classification Models

**Goal:** Predict whether a patient is diabetic or non-diabetic.

---

### 3. Email Spam Detection
Build a model to classify emails as Spam or Not Spam.

**Techniques Used:**
- Text Preprocessing
- CountVectorizer / TF-IDF
- Natural Language Processing (NLP)
- Naive Bayes Classification

**Goal:** Classify emails into:
- Spam (1)
- Not Spam (0)
